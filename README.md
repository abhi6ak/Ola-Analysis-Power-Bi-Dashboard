# Ola-Analysis-Power-Bi-Dashboard
Ola Cabs Operational Analysis Dashboard 🚖 This repository contains a comprehensive Power BI Dashboard that analyzes Ola Cabs' booking data for January 2025. The project focuses on key performance indicators (KPIs) such as booking success rates, revenue distri, vehicle performance, and cancellation reasons to provide actionable business insights.

📊 Project Overview
The goal of this project was to transform raw ride-hailing data into a multi-page interactive report. It allows stakeholders to monitor operational health, understand customer behavior, and identify friction points in the booking process.

🚀 Key Features & Insights
1. Vehicle Type Performance
Segment Analysis: Detailed breakdown of categories including Prime Sedan, Prime SUV, Prime Plus, Mini, Auto, Bike, and E-Bike.

Revenue vs. Distance: Tracking total booking value against total distance traveled to identify the most profitable vehicle segments.

Consistency: Data shows a remarkably even distribution of booking values across all vehicle types (approx. 3.7M to 3.9M each).

2. Revenue & Payment Trends
Payment Preferences: Visualizes the dominance of Cash payments (reaching over 6.8M) compared to Credit Cards, UPI, and Wallets.

Daily Trends: A time-series analysis of ride distances to monitor peak demand days throughout January.

3. Cancellation & Operational Health
Funnel Metrics: Tracks the journey from 100,000 Total Bookings to 62,085 Succeeded Bookings.

Customer Friction: Analyzes why customers cancel, with "Driver is not moving," "Wrong Address," and "AC not working" being top contributors (each ~20%).

Driver Analysis: Monitoring cancellation rates from the driver's side to ensure platform reliability.

🛠️ Tools Used
Power BI Desktop: For data modeling, DAX measures, and report design.

Data Cleaning: Handled missing values (e.g., incomplete cancellation data) and data type formatting.

DAX (Data Analysis Expressions): Used for calculating success rates and total booking values.

📂 How to Use
Download the .pbix file from this repository.

Open it in Power BI Desktop.

Use the sidebar navigation (Overall, Vehicle Type, Revenue, Cancellation, Ratings) to toggle between different views.
